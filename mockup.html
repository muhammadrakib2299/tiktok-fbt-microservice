The merchant never has to dig for a problem — problems come to them.

Modal: connect a shop

Connect a TikTok FBT shop

You’ll be redirected to TikTok to authorise us. We only request scopes needed for FBT — product, inventory, orders, fulfilment, finance.

Interstitial: link to seller (optional)

Link to a seller account?

Linking shows side-by-side stock (own warehouse + FBT) in your dashboards. Optional; you can change this any time.


What happens behind the scenes: OAuth code → token exchange → AES-256-GCM encrypt → insert fbt_accounts row → publish tiktok_fbt.to.wms.account_connected → core-service creates one virtual warehouse row (warehouse_type = 'fbt_tiktok') → first inventory poll queued. All inside ~3 seconds.
